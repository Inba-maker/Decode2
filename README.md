# Basic Encryption and Decryption using Caesar Cipher

## Project Overview

This project demonstrates a simple Encryption and Decryption system using the Caesar Cipher technique. It allows users to encrypt a message using a shift key and decrypt it back to its original form.

The Caesar Cipher is one of the oldest and simplest encryption methods where each letter is shifted by a fixed number of positions in the alphabet.

## Objectives

- Understand basic encryption concepts
- Implement Caesar Cipher logic
- Encrypt user input text
- Decrypt encrypted text
- Display both encrypted and decrypted outputs

## Features

- Encrypts alphabetic characters
- Preserves spaces and symbols
- Supports uppercase and lowercase letters
- Decrypts encrypted text correctly
- User-defined shift value

## Requirements

- Python 3.x

## How the Program Works

### Encryption

Each letter is shifted forward by a specified number.

Example:

Original Text:
HELLO

Shift:
3

Encrypted Text:
KHOOR

### Decryption

The encrypted text is shifted backward by the same number.

Encrypted Text:
KHOOR

Shift:
3

Decrypted Text:
HELLO

## Sample Output

Enter the text:
Cyber Security

Enter the shift value:
3

Encrypted Text : Fbehu Vhfxulwb

Decrypted Text : Cyber Security
